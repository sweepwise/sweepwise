Cleanium

A macOS menu-bar disk cleaner that explains every candidate in plain English — what it is, how risky it is, and how to get it back — before you delete anything.

How it works

1

Scan

Cleanium walks the folders you choose and matches what it finds against 29 built-in rules for caches, build leftovers, AI model stores, and stale downloads. Results stream in live — pause anytime and act on what's been found so far.

2

Understand

Every candidate shows its full path, a plain-English explanation, a risk grade, and exactly how to get it back if you ever need it. No mystery checkboxes.

3

Clean

Everything goes to the Trash by default, so nothing is gone until you say so. Permanent delete is a separate, explicit choice for items over 2 GB.

Why it's different

Explains, never guesses for you

Other cleaners show a number and a checkbox. Cleanium tells you what each folder is and what deleting it costs you — then lets you decide.

AI that uses what you already pay for

Unknown large folders can be explained by your locally installed claude, codex, or gemini CLI. Your subscription, your machine — no API keys, and Cleanium itself never touches the network.

Learns only with consent

When the AI classifies something you delete, Cleanium offers to remember the rule. Nothing is saved without your explicit approval — you're asked every time.

Built for developers' disks

node_modules, DerivedData, cargo and gradle caches, simulator images, Ollama and Hugging Face models — the multi-gigabyte stuff other cleaners walk past.

Open source, no dependencies

Pure Swift and SwiftUI, zero third-party packages, MIT licensed. Read every line before you trust it with your disk.

Honest about cost

AI calls show their token usage — input, cached, and output — right in the popover. Folders are batched so you pay the CLI overhead once, not per folder.

Install

Unzip, drag Cleanium.app to Applications, then right-click → Open the first time.

Cleanium is ad-hoc signed, not notarized, so macOS asks for that one explicit first open. Prefer building it yourself? git clone the repo and run ./scripts/bundle.sh — no Xcode project needed.
